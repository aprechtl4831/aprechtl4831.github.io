Now that we have a row for each user we need to populate the values. Currently the new rows have nan for all the values, we will fix this by using a moving average of sorts. The value of the new row will be the average of the previous and the next value. If the previous value is non-existent we set the new value to the next value. If the next value is non-existent or nan we set the new value to the previous value.

In[]:
# Iterate through each column (starting from the third column)
for col in dailyActivity.columns[2:]:
    for index, value in enumerate(dailyActivity[col]):
        if pd.isna(value):
            prev_value = dailyActivity.at[index - 1, col] if index > 0 else None
            next_value = dailyActivity.at[index + 1, col] if index < len(dailyActivity) - 1 else None
            
            if pd.isna(prev_value) and pd.isna(next_value):
                continue
            elif pd.isna(prev_value):
                dailyActivity.at[index, col] = next_value
            elif pd.isna(next_value):
                dailyActivity.at[index, col] = prev_value
            else:
                dailyActivity.at[index, col] = (prev_value + next_value) / 2

# Reset index
dailyActivity.reset_index(drop=True, inplace=True)

Exploratory Data Analysis

EDA involves visually and statistically analyzing data to understand its characteristics and patterns. It helps identify trends, outliers, and relationships within the data, guiding further analysis and hypothesis generation. The first analysis will be a comparison of user groupings to sleep levels and heart rate levels. Groups will be formed based on activity level. We will split the users into three groups: low activity, medium acitivty, and high activity. The average sleep level will be plotted against time in an attempt to see which group sleep the best. There are many studies which link productivty to sleep. Readings on the subject can be found here. The code below makes the grouping, averages the sleep for each day and plots the groups against time.

In[]:
# Calculate the sum of 'VeryActiveDistances' for each user
user_sum_dist = dailyActivity.groupby('Id')['VeryActiveDistance'].sum()

# Determine the thresholds for low, medium, and high distances
thresholds = user_sum_dist.quantile([1/3, 2/3])

# Function to categorize users based on distance
def categorize_distance(dist):
    if dist <= thresholds.iloc[0]:
        return 'Low'
    elif dist <= thresholds.iloc[1]:
        return 'Medium'
    else:
        return 'High'

# Apply the categorization function to each user
user_sum_dist_category = user_sum_dist.apply(categorize_distance)

# Count the occurrences of each category
category_counts = user_sum_dist_category.value_counts()

# Determine the minimum count for each category to split evenly
min_count = min(category_counts)

# Group users by category
low_group = user_sum_dist_category[user_sum_dist_category == 'Low'].index[:min_count]
medium_group = user_sum_dist_category[user_sum_dist_category == 'Medium'].index[:min_count]
high_group = user_sum_dist_category[user_sum_dist_category == 'High'].index[:min_count]

# Convert 'DateColumn' to datetime objects
dailySleep['SleepDay'] = pd.to_datetime(dailySleep['SleepDay'])

# Create DataFrames for each group
lowSleep_df = dailySleep[dailySleep['Id'].isin(low_group)]
mediumSleep_df = dailySleep[dailySleep['Id'].isin(medium_group)]
highSleep_df = dailySleep[dailySleep['Id'].isin(high_group)]

# Calculate the average Sleep for each group
low_avg = lowSleep_df.groupby('SleepDay')['TotalMinutesAsleep'].mean().tolist()
medium_avg = mediumSleep_df.groupby('SleepDay')['TotalMinutesAsleep'].mean().tolist()
high_avg = highSleep_df.groupby('SleepDay')['TotalMinutesAsleep'].mean().tolist()

# Plotting
plt.figure(figsize=(10, 6))  # Adjust the figure size as needed

# Plot the data
